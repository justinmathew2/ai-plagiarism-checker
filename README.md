# 🎓 AI Plagiarism Checker

An AI-powered academic assistant that detects plagiarism, identifies AI-generated content, and suggests improvements to reduce plagiarism in documents.

---

## 🚀 Features

* 📄 Upload PDF/DOCX files
* 🔍 Basic plagiarism detection
* 🤖 AI-generated content detection
* 🧠 Smart rewrite suggestions
* 📊 Clean dashboard with results

---

## 🛠️ Tech Stack

* Frontend: React (Google Stitch)
* Backend: FastAPI (Antigravity)
* AI: OpenAI APIs

---

## ⚡ How It Works

1. Upload your document
2. Text is extracted and analyzed
3. Plagiarism & AI scores are generated
4. Suggestions are provided to improve originality

---

## ⚠️ Disclaimer

This tool provides basic plagiarism detection and AI assistance.
It is not a replacement for professional tools like Turnitin.
